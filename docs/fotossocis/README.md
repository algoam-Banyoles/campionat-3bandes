# Fotos dels socis

Aquesta carpeta conté **dades privades** (fotos dels socis) i **NO** s'ha de pujar al repositori públic.

Convenció:

- Nom del fitxer: `<numero_soci>.jpg` (p.ex. `8747.jpg`)
- Format preferit: JPEG optimitzat (~800×800 px, qualitat 85)
- Les fotos es publiquen a un bucket privat de **Supabase Storage** (`socis-fotos`) i només els administradors hi poden accedir des de l'app.

Les imatges d'aquesta carpeta estan excloses via `.gitignore` (excepte aquest README i el `.gitkeep`).
